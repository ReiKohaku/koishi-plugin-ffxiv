# koishi-plugin-ffxiv

一款《最终幻想14》辅助插件。

## 功能列表/TODO列表

已实现功能：
* 查询板子物价（基于universalis的API）
* 查询艾欧泽亚时间（ET）
* 闹钟（仿照游戏内闹钟）
* 扔骰子
* 抽签（今日艾欧泽亚运势）
* 服务器状态查询
* 订阅官网新闻（当前仅支持国服）
* 艾欧泽亚图鉴
